> 사용자의 게임 화면을 실시간으로 분석해 보스 체력과 패턴 변화를 감지하고, 다음 패턴 발생 시점을 예측하는 웹 서비스입니다.

---

<img src="https://img.shields.io/badge/My_Role-ab47bc?style=for-the-badge" />

- Web API로 화면 픽셀을 추출, 패턴 예측 음성 알림
- Web Worker로 데이터 병렬 분석
- Sobel 필터 등 이미지 프로세싱으로 관심 영역 추출
- 7세그먼트 영역 이진화 후 각 세그먼트 비트 추출로 숫자 인식
- BFS로 영역 분리 후 비트 패턴 매칭
- 색상 양자화 후 특정 색상 비율 초과 시 패턴 발생 감지
